# 「自分のレシピ」タブを「レシピノート」にリネーム

## 目的・背景

現在「自分のレシピ」というタブ名は、ユーザーが自分で作成・登録したレシピの一覧という
ニュアンスを持つが、実態は **AI が生成したレシピが auto-save された一覧** であり、
ユーザーが直接書いたものではない（空状態メッセージ:
`レシピを生成すると自動で保存されます`）。

アプリシンプル化（`docs/plans/archive/app-simplification.md`）で「みんなのレシピ」
タブと「いいね」機能が撤去され、`liked === 1` フィルタも外れたことで、
このタブは事実上 **生成済みレシピの履歴 / 蓄積場所** になった。

そこで、`本のように貯まっていく` メタファーが auto-save 挙動と整合し、内部 docs
（`web/admin/app.js` の "レシピブック"）やスクショ撮影計画でも近い呼称が使われていた
ことから、ユーザーとの議論を経て **「レシピノート」** に改称する。

## 対応方針

タブラベルを変更し、外向き文書（README / about.html）や admin 内の説明文で
"自分のレシピ" を参照している箇所を新名称に揃える。アーカイブ済プラン
（`docs/plans/archive/`）と `DONE.md` は履歴なので触らない。

## 影響範囲

### 必ず変える（本タスクのスコープ）
- `mobile/app/(tabs)/_layout.tsx:58` — `title: '自分のレシピ'` を `'レシピノート'` に
- `README.md:18` — タブ説明の "みんなのレシピ / 自分のレシピ" を現状（`レシピノート`
  単独）に更新。アプリシンプル化以降「みんなのレシピ」は撤去済みなのでこの行は
  README として既にズレているため併せて整理する
- `README.md:230` — API 表 `GET /api/saved-recipes` 説明の "自分のレシピ一覧" を
  "レシピノート一覧" に
- `web/about.html:409` — `<img ... alt="自分のレシピ画面">` を `alt="レシピノート画面"` に

### 検討して触る（admin の説明文）
- `web/admin/app.js:978` — `"左: みんなのレシピ / 中: 自分のレシピ / 右: メニュー(固定)"`
- `web/admin/app.js:2242` — `みんなのレシピ / 自分のレシピ（タブ切替）` 行
- `web/admin/app.js:2258` — ASCII ツリー内 `# レシピブック` コメント
- `web/admin/app.js:2368` — `レシピ機能 — ... レシピブック（みんな/自分）`

これらは admin 画面の説明文で、いずれも `みんなのレシピ` がまだ存在する前提で
書かれており、シンプル化の時点で既に不整合。本タスクでは **"自分のレシピ" を
"レシピノート" に置換するだけ** に絞り、`みんなのレシピ` の残存整理は
本タスクの後段ステップとして TODO に切り出す（混ぜると差分が膨らむため）。

### 触らない
- `docs/plans/archive/*.md` — アーカイブ済プランは当時の文脈で凍結
- `DONE.md` — 完了履歴
- `mobile/app/(tabs)/recipes.tsx` 本体 — "自分のレシピ" 文字列を含まない
  （検索プレースホルダは `レシピを検索...`、空状態は
  `レシピを生成すると自動で保存されます` のままで自然）
- `mobile/__tests__/` — "自分のレシピ" を含むテストは無し（grep 確認済）
- API パス・DB スキーマ・サーバ実装 — `/api/saved-recipes` のままで変更しない
  （UI ラベルだけ変更し内部識別子は据え置く）
