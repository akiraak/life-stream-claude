# 買い物リスト機能 仕様書

## 概要

日常の買い物管理を支援するシンプルなリスト機能。
Web ブラウザおよび iOS アプリから利用可能。

## 機能一覧

| 機能 | 説明 |
|------|------|
| アイテム追加 | 名前を入力してリストに追加 |
| アイテム削除 | 個別に削除 |
| チェック | 購入済みアイテムをチェックし非表示にする |
| 一覧表示 | 未チェックのアイテムのみ表示 |

## API エンドポイント

### `GET /api/shopping`
未チェックのアイテム一覧を取得する。

### `POST /api/shopping`
新しいアイテムを追加する。

```json
{ "name": "牛乳" }
```

### `PUT /api/shopping/:id`
アイテムの状態を更新する（チェック等）。

### `DELETE /api/shopping/:id`
アイテムを削除する。

## レスポンス形式

全 API 共通のレスポンス形式:

```json
{
  "success": true,
  "data": { ... },
  "error": null
}
```
