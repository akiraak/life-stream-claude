買い物リスト機能 仕様書

日常の買い物管理を支援するシンプルなリスト機能

機能一覧

機能説明
アイテム追加名前を入力してリストに追加
アイテム削除個別に削除
チェック購入済みアイテムをチェックし非表示にする
一覧表示未チェックのアイテムのみ表示

API エンドポイント

GET /api/shopping

未チェックのアイテム一覧を取得する。

POST /api/shopping

新しいアイテムを追加する。

{ "name": "牛乳" }

全 API 共通のレスポンス形式: { success, data, error }
